# Draft Session Plan

## Purpose

Turn the current planning brief and synthesized campaign context into a usable draft session structure.

## Inputs

- `campaign_ref`
- active `plan_ref`
- current planning brief
- synthesized context

## Outputs

- updated session-planning Working Record with a draft plan

## Dependencies

- [WORKING_RECORD.md](/Users/durdin/Projects/Durdin/tal-dore-tales/SPECIFICATIONS/WORKING_RECORD.md)

## Files Read

- active session-planning Working Record
- supporting campaign context already identified for the plan

## Files Written

- updated session-planning Working Record

## Approval

- none

## Constraints

- the draft must separate anchors from optional ideas
- the draft should include why the session matters now
- scene ideas, clue paths, NPCs, and combat support should fit the stated campaign direction
- contradictions, overload, or weak hooks should be surfaced plainly

## Failure

- draft ignores the human brief
- draft overloads the session without warning
- optional ideas are presented as required truths
- unsupported continuity assumptions are hidden
