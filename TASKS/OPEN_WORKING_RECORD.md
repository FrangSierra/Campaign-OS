# Open Working Record

## Purpose

Create a Working Record when ambiguity or contradiction blocks safe Event proposal.

## Inputs

- `campaign_ref`
- `session_ref`
- unresolved issue

## Outputs

- optional Working Record files

## Dependencies

- [WORKING_RECORD.md](/Users/durdin/Projects/Durdin/tal-dore-tales/SPECIFICATIONS/WORKING_RECORD.md)

## Files Read

- current Session
- supporting Source Records
- affected canon

## Files Written

- Working Record files in `WORKING/`

## Approval

- none

## Constraints

- speculation remains distinct from canon
- affected records and evidence are linked

## Failure

- ambiguity exists but is not preserved
- unsupported speculation is treated as truth
