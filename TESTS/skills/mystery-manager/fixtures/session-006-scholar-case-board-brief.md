# Mystery Manager Fixture - Session 6 Scholar Case

## Status

Skill rehearsal fixture for `mystery-manager`.

## Purpose

This fixture asks the skill to turn the planned Session 6 scholar case into a disciplined mystery board.

## Inputs

- use the approved Session 6 plan as the mystery-design anchor
- use current campaign mystery state so the first Emon case does not get swallowed by Arven or the Time Agency
- use the latest QA and prep outputs as reveal guardrails

## Requested Output

Produce a mystery board that keeps these layers separate:

- what Emon publicly believes
- what the party could learn in Session 6
- what clues may mislead without being unfair
- what deeper truth options exist behind the staged death

## Success Criteria

- keep the case grounded and human-scale
- avoid locking in one master conspiracy answer
- keep Ivy and Brann useful but non-solution-bearing
- preserve the difference between designed possibility and established truth
