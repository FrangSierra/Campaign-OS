# Campaign Health Fixture

## Status

Non-canonical test fixture for Skill rehearsal only.

## Purpose

This fixture tests whether `campaign-health` can:

- read the campaign after Session 5 and before Session 6
- distinguish stable strengths from emerging stress points
- use recent QA, prep, mystery, and consequence outputs without treating them as canon
- produce a useful maintenance snapshot rather than a contradiction report

## Prompt

Build a campaign health snapshot for `echoes-of-tal-dorei` immediately before the first true Emon session.

The snapshot should address:

- whether the campaign is stable after the prologue reset
- whether Emon is ready to function as a compelling home base
- whether the active mystery load is balanced
- whether the recurring cast and consequence pressure are healthy

## Expected Boundary

- read-only rehearsal output only
- soft warnings should remain clearly softer than contradictions
- no canon, state, or working writes
