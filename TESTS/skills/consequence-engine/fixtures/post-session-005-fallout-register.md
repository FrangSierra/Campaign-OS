# Consequence Engine Fixture

## Status

Non-canonical test fixture for Skill rehearsal only.

## Purpose

This fixture tests whether `consequence-engine` can:

- derive delayed fallout from approved Session 5 canon without inventing off-screen events
- use current Session 6 prep and QA warnings to decide what is ready to surface next
- separate already-established consequences from future-facing pressure
- keep all outputs non-canonical and prep-facing

## Prompt

Using the current `echoes-of-tal-dorei` campaign context, build a consequence register after Session 5 and before Session 6.

Focus on:

- what fallout is already canon because Session 5 established it
- what delayed consequences are now plausible but not yet real
- which candidates are best suited for Session 6 or shortly after
- what should stay dormant so Emon can establish its own grounded identity first

## Review Lens

The rehearsal should prove that the skill can keep the following pressures separate:

- Arven and the Time Agency as deep campaign fallout
- the restored world's public stability
- the first under-Emon civic mystery
- new recurring NPCs such as Brann and Ivy

## Expected Boundary

- read-only rehearsal output only
- no `CANON/` or `STATE/` writes
- no claim that an unplayed consequence has already happened
