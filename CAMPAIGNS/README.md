# Local Campaign Data

This directory is where live campaigns belong. It is intentionally ignored by Git so that session notes, player information, setting references, and private prep do not end up in the public repository.

Create one folder per campaign, for example `CAMPAIGNS/my-campaign/`. The [Campaign Bootstrap command](../COMMANDS/BOOTSTRAP_CAMPAIGN.md) can create a safe starting structure from a folder of existing material or a short guided interview.

Do not add a live campaign directory to Git unless you deliberately want to publish it.
