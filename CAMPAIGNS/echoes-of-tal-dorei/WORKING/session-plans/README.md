# Session Plans

This directory stores future-session planning Working Records for `campaign-echoes-of-tal-dorei`.

These records are:

- non-canonical
- future-facing
- safe to revise repeatedly
- commit-able by explicit human approval without changing canon

Use this directory for:

- session hooks
- scene structure drafts
- NPC candidates
- continuity warnings
- revision history for planned sessions

Do not store reusable setting or rules references here.

Those belong in:

- `CAMPAIGNS/<campaign-id>/REFERENCE/` for lore and setting primers
- `CAMPAIGNS/<campaign-id>/RULES/<system-id>/` for system-specific planning references

Approved plans stay here as non-canonical prep artifacts.

That means:

- before play, they are the current best draft
- during play, they are a guide, not a script
- after play, they remain useful as historical prep context
- actual canon still comes only from post-session notes and the normal Truth Pipeline

Do not treat anything here as established world truth until it happens in play and later enters the normal Truth Pipeline.
